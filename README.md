# kimat
This is a PHP based application where users can upload an image of a receipt of their groceries fand get a comparison of everything they've purchased to see if they are getting a good deal.

## Getting Started

Step 1.  Git clone the repository</br>
Step 2.  Make sure you have [PHP](https://www.php.net/manual/en/install.php) installed.<br>
Step 3.  Make you have a web server installed.<br>
Step 4.  TBD<br>

## Using kimat

Upload your receipt to kimat.  Kimat will return what it detects in your receipt.  Check the items it has correctly identified and it will return what it knows as the current best price per unit for that grocery item.
